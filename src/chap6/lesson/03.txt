次のコードがあります。

class Parent {
    int num = 100;
    void show() {
        System.out.println(num);
    }
}
class Child extends Parent {
    int num = 500;
}
public class Main {
    public static void main(String[] args) {
        Child c = new Child();
        c.show();
    }
}

コンパイル、実行した結果として正しいものは次のどれですか。1つ選択してください。

A. 100
B. 500
C. コンパイルエラー
